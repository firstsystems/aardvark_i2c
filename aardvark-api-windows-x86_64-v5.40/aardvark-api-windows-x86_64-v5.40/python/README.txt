                  Total Phase Aardvark Sample Code
                  --------------------------------

Introduction
------------
This directory contains examples that use the Aardvark Rosetta Python
language bindings.


Contents
--------
See top level EXAMPLES.txt for descriptions of each example.


Instructions
------------
The Total Phase API bindings support Python 2.6+ and 3.5+.  The sample code
has been written specifically for these versions.  There is no sample code
available for other versions of Python.

1) Download Python from:
http://www.python.org/

2) Install the Python package

3) Open a command shell and change to the examples directory

4) Type: 'python3 SCRIPT'
example: python3 aadetect.py
